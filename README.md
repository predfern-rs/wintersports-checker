# Majestic Winter Sports Filter (Streamlit)

Upload a Majestic CSV and extract all rows where TopicalTrustFlow_Topic contains:

**Sports/Winter Sports**

The output includes:
- Item
- TopicalTrustFlow_Topic
- TopicalTrustFlow_Value
- Topic_Index

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
